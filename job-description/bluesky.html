2024-02-11: Backend Developer - Bluesky

Bluesky’s mission is to transition the social web from platforms to protocols. We’re building a federated social network where users have more power. Our team has decades of combined experience building distributed applications. As a backend dev, you’ll be writing protocol and backend application code that will be a critical part of transforming social media.

Responsibilities:

  • Scale our existing services to millions of users
  • Develop and maintain the AT Protocol implementation
  • Create SDK and API tools for an ecosystem of protocol developers

You might be a good fit if you:

  • Have experience with Typescript and/or Go
  • Have a strong knowledge of data structures and software design principles
  • Have experience working with Postgres at scale
  • Have experience with Scylla/Cassandra or Elasticsearch
  • Are familiar with distributed systems
  • Have built scalable and high-performance server-side applications
  • Like working on small, fast-moving teams
  • Have read the atproto docs and want to contribute

We’re especially excited if you have:

  • Previous experience working on social sites
  • Experience with cryptographic primitives such as public/private keys and merkle trees
  • Experience with the IPLD or DID ecosystems
